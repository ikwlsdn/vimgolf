# vimgolf

### - Add quotes to ansible playbook
입력 : GWi"<End>"<Esc>ZZ

  G : 마지막 줄로 이동
  
  W : {앞으로 이동
  
  i : 입력모드 시작 
  
  END : 입력모드에서 줄의 마지막으로 이동
    
  ZZ : :wq와 동일한 기능(버튼수가 하나 더 적음)
