# vimgolf

### 1) Add quotes to ansible playbook
입력 : GWi"<End>"<Esc>ZZ

  G : 마지막 줄로 이동
  
  W : {앞으로 이동
  
  i : 입력모드 시작 
  
  END : 입력모드에서 줄의 마지막으로 이동
    
  ZZ : :wq와 동일한 기능(버튼수가 더 적음)

  ### 2) simple replacements
  입력 : %s/sublime\|emacs/vim/g<CR>ZZ
  
  %s : 치환 명령어
  
  /sublime\|emacs : sublime 또는 eamcs 문자열
  
  /vim : vim문자열로 치환
  
  /g : 모든 줄에 적용
  
  ZZ : :wq와 동일한 기능(버튼수가 더 적음)
